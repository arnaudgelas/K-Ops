# Source Summary: src-fac0000001

## Summary

Torque v2.0.0 reached general availability on 2026-01-15. It is a Rust
stream-processing engine maintained by Torque Labs and released under the
Apache-2.0 license.

## Key Claims

- v2.0.0 was released on 2026-01-15 as the first generally available (GA) build.
- v2.0.0 introduces exactly-once processing semantics for windowed aggregations.
- Exactly-once is enabled by setting `processing.guarantee = exactly_once` in
  the `torque.toml` configuration file; the default remains `at_least_once`.
- Torque v2.0.0 is licensed under Apache-2.0.

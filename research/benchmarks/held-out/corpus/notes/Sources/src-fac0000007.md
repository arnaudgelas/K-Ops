# Source Summary: src-fac0000007

## Summary

Torque Labs published a first-party benchmark for v2.1.0.

## Key Claims

- Torque sustains 1.2M events/sec on a single 8-core node in the vendor test.
- The workload is a 5-second tumbling-window aggregation over synthetic events.
- The report is authored and run by Torque Labs (vendor self-benchmark).

## Evidence Notes

- Vendor-run benchmark; not independently reproduced. This is the ORIGINAL
  primary from which two secondary blogs (src-5ec0000012, src-5ec0000013)
  derive their numbers.

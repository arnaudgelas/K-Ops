# Source Summary: src-fac0000004

## Summary

TQ-SA-2026-001 discloses a deserialization remote-code-execution (RCE)
vulnerability in Torque, published 2026-04-02.

## Key Claims

- The vulnerability is a deserialization RCE with CVSS base score 9.1 (critical).
- Affected versions: 2.0.0 through 2.1.0 inclusive.
- The vulnerability is fixed in v2.1.1; all users must upgrade immediately.

## Evidence Notes

- Official security advisory from Torque Labs; organization: Torque Labs.
